แชร์ตู้เย็นในบ้าน (Family Cloud Sync via Firebase)

ซิงก์ข้อมูลวัตถุดิบและของในตู้เย็นแบบเรียลไทม์ (Real-time) กับโทรศัพท์ของทุกคนในครอบครัวผ่านฐานข้อมูลส่วนตัวของคุณ

รหัสแชร์ในบ้าน:
🔒
ระบบคลาวด์ส่วนตัว Google Firebase (เชื่อมต่อง่าย ไม่ต้องใช้ Config ซับซ้อน) เพื่อตัดขั้นตอนยุ่งยากและปัญหาสลับรหัสโครงการ ทางเราได้อัปเดตให้เชื่อมต่อผ่านระบบ REST API ของเบราว์เซอร์โดยตรง **โดยคุณไม่จำเป็นต้องนำโค้ดตั้งค่ายาวๆ มาวางอีกต่อไป** ใช้เพียงแค่ลิงก์ฐานข้อมูล (Database URL) จุดเดียวเท่านั้นครับ!
ขั้นตอนการเชื่อมต่อฐานข้อมูล Firebase ของคุณ (ใช้เวลาไม่ถึง 1 นาที):
  1. เปิดเว็บเบราว์เซอร์ไปที่: https://console.firebase.google.com
  2. สร้างโครงการของคุณเอง จากนั้นกดเมนูด้านซ้าย **Build** > **Realtime Database** และกดปุ่ม **Create Database**
  3. สำคัญ: ในขั้นตอนสิทธิ์การเข้าถึง (Security Rules) ให้เลือก **Start in test mode** (เริ่มต้นในโหมดทดสอบ) เพื่อให้แอปเขียนอ่านได้ และกด **Enable** (หรือในแท็บ Rules ด้านบนเป็น `.read: true` และ `.write: true` แล้วกด Publish)
  4. ในหน้า Realtime Database คัดลอกลิงก์ตรงแถบสีขาวด้านบนสุด (เช่น https://[ชื่อโครงการ]-default-rtdb.firebaseio.com/) มาวางใส่ในช่องด้านล่างนี้ได้เลยครับ!
